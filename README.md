# Diary DevOps Project

## Description
A simple diary application demonstrating a full DevOps pipeline:
Flutter frontend + backend API + Docker + Kubernetes + CI/CD + Monitoring.

## Architecture
Frontend (Flutter) → Backend (API) → Containerized → Kubernetes deployed

## Features
- Create diary entries
- View entries
- Delete entries

## DevOps Tools
- GitHub Actions (CI/CD)
- Docker
- Kubernetes
- ArgoCD
- Prometheus & Grafana
